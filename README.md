## Setup NodeJS project with Typescript and Jest

```
  npm init -y
  // Setup typescript
  npm i -D typescript ts-node @types/node @tsconfig/node22
  // Setup jest
  npm i -D jest ts-jest @types/jest
  // To run test 
  npm test
```

```
 // Clone this repo
 git clone ...
 // install npm packages
 npm i
 // Test
 npm test
```
